# Objektinio programavimo laboratoriniai darbai

VU ISI 1k. 2sem. Objektinio programavimo laboratoriniai darbai.

## Apie programą

Studentų duomenų bazė. Studentų duomenys įvedami tol, kol į vardo ir pavardės eilutę neįrašoma "stop stop". Maksimalus studentų namų darbų įvesties skaičius yra 10, tačiau šią įvestį galima sustabdyti ir anksčiau, įrašius "-1" kaip namų darbų pažymį.
